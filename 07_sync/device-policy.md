# Device Policy

## Mac

- Primary human editing device.
- Primary Obsidian configuration device.
- Preferred place for long writing and manual review.

## Windows

- Reading, clipping, and light editing.
- Avoid bulk renames.
- Avoid Windows-incompatible filenames.

## Linux Server

- Agent runner.
- Backup and Git snapshot node.
- Scheduled lint and context export node.
- Should avoid bulk writes unless no other device is actively editing.
