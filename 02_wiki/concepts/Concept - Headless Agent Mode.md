# Concept - Headless Agent Mode

## Summary

Headless agent mode means Codex or another command-line agent can maintain the knowledge base without Obsidian installed locally.

## Key Points

- The storage layer is plain Markdown.
- Obsidian is a visual editor and browser, not the source of truth.
- Agents can search, edit, link, summarize, and export knowledge directly from the filesystem.
- Query quality depends on maintained Markdown, index pages, backlinks, source notes, and stable filenames.
