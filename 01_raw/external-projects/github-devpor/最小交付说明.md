# github-devpor 最小交付说明

## 1. 交付目标

当前最小可交付版本聚焦以下能力：

- GitHub 仓库搜索与 README 拉取
- 下载最小链路（开始、进度、取消）
- 支付 Mock 最小链路（创建订单、状态查询、模拟回调、验单）
- 付费模式下安装执行前置扣次校验
- 远程服务端守护与日志轮转（pm2 + pm2-logrotate）

## 2. 本地运行步骤

1. 安装依赖：
   - `npm install`
2. 本地开发：
   - `npm run dev`
3. 构建：
   - `npm run build`
4. 测试：
   - `npm test`
